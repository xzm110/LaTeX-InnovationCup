# LaTeX-InnovationCup
这是中国人民大学“创新杯”学生学术科技作品竞赛 LaTeX 模板。

是我（缝合怪兼 LaTeX 业余爱好者）google bing 写的统计学院模板。

编辑器：TeXLive 2020 套装

编译引擎：XeLaTeX

## 使用方法
将 tex 文件和 imgs 文件夹放至同一目录，编译即可。
